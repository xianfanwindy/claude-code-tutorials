# 01. 认识 Claude Code

Claude Code 是 Anthropic 官方的命令行智能编程助手。它运行在你的终端中，可以读取项目文件、理解代码结构、编辑文件、运行测试、调用 Git、创建 PR，并通过权限机制让用户控制每一步操作。

## Claude Code 适合做什么

### 代码理解

- 解释项目结构
- 查找某个功能在哪里实现
- 梳理调用链
- 总结模块职责
- 阅读大型 diff 或 PR

### 代码修改

- 修复 bug
- 添加小到中型功能
- 重构局部代码
- 编写测试
- 更新配置
- 修改文档

### 工程协作

- 生成提交信息
- 创建 Pull Request
- 跟踪 CI 状态
- 审查当前 diff
- 做安全审查
- 根据 issue 或需求拆解任务

### 自动化工作流

- 在保存后运行格式化
- 在提交前运行检查
- 定期轮询部署状态
- 使用 MCP 访问 GitHub、数据库、监控系统或内部工具

## Claude Code 的核心特点

### 1. 在真实项目中工作

Claude Code 不是单纯的聊天机器人。它可以在本地工作目录中执行工程操作，例如读取文件、编辑文件、运行命令和查看 Git 状态。

### 2. 用户掌控权限

敏感操作通常需要用户确认，例如运行命令、写文件、推送代码等。你可以逐步授权，也可以配置更细的允许列表。

### 3. 能理解上下文

Claude Code 会结合当前对话、项目文件、CLAUDE.md、记忆系统和工具结果来完成任务。

### 4. 支持工程级工作流

它不仅能写代码，还能参与计划、验证、review、提交、PR 和排错。

## 和普通聊天式 AI 的区别

| 能力 | 普通聊天 AI | Claude Code |
| --- | --- | --- |
| 读取本地项目 | 通常不能 | 可以 |
| 修改文件 | 通常不能 | 可以 |
| 运行测试 | 通常不能 | 可以 |
| 创建 Git 提交 | 通常不能 | 可以，需授权 |
| 使用项目规范 | 需要手动粘贴 | 可读取 CLAUDE.md |
| 长期偏好 | 弱 | 可用 memory 保存 |
| 调用外部工具 | 有限 | 可通过 MCP 扩展 |

## 典型使用方式

你可以像这样给 Claude Code 下达任务：

```text
解释这个项目的目录结构
```

```text
找出登录流程在哪里实现，不要修改代码
```

```text
修复用户退出登录后仍显示头像的问题，并运行相关测试
```

```text
为当前改动创建一个提交，但不要 push
```

```text
审查当前 diff，重点关注安全问题
```

## 使用心态

把 Claude Code 当成一个会读代码、会执行命令、会写补丁的协作者，而不是一个完全自动驾驶系统。

你需要提供：

- 明确目标
- 约束条件
- 验收标准
- 风险偏好
- 是否允许推送、部署或修改共享资源

Claude Code 负责：

- 探索代码
- 提出方案
- 执行修改
- 运行验证
- 总结结果和风险

## 不适合直接交给 Claude Code 的任务

以下任务需要额外谨慎：

- 删除大量文件或数据库
- 生产环境部署
- 处理真实用户凭据
- 未授权的安全测试
- 大规模外部扫描
- 绕过安全检测
- 修改组织级权限

这些任务不是绝对不能做，而是必须有明确授权、边界和回滚方案。
